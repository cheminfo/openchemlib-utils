getConnectivityMatrix

src/util/getConnectivityMatrix.js

Returns a connectivity matrix

getConnectivityMatrix(molecule: OCL.Molecule, options: object)
Parameters
molecule (OCL.Molecule)
options (object = {})
Name Description
options.pathLength boolean (default false) get the path length between atoms
options.mass boolean (default false) set the nominal mass of the atoms on diagonal
options.atomicNo boolean (default false) set the atomic number of the atom on diagonal
options.negativeAtomicNo boolean (default false) set the atomic number * -1 of the atom on diagonal
options.sdt boolean (default false) set 1, 2 or 3 depending if single, double or triple bond
options.sdta boolean (default false) set 1, 2, 3 or 4 depending if single, double, triple or aromatic bond
